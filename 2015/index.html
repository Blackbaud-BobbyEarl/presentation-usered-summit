Sky UX

Sky UX is Blackbaud’s next-generation user-experience framework. It provides a design philosophy to create a consistent, cohesive experience for Blackbaud services and a toolset to implement the design philosophy.

(Thanks John!)

Sky UX is the new black

  • UX, visual, and engineering components shared across entire organization
  • Feedback and contributions are expected internally and externally.
  • That's right, we're going Open Source!

How we did it, what makes Sky UX different

  • Not just source in GitHub, but issues (our backlog) too.
  • Technical embedded in code and automatically generated (Using Stache)
  • Release frequently with confidence
    • Using Travis CI to consistently run automation tests
    • Run all our tests in all the browsers
    • Visual regression testing - compare screenshots for each component on every release

What's Next for Sky UX

  • Crossing our T's and dotting our I's to go public.
  • Continue contributing components for NXT and other products.
  • Officially transferring to 100% Sky UX Team after BBCON.

Sky UX is not owned by the Sky team - it's owned by all teams!
