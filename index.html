What is GitHub?

GitHub = Git + Hub

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

Think Team Foundation Server, but distributed and free

Powerful collaboration, code review, and code management for open source and private projects.

On top of git capabilities, GitHub provides:

  • Central location for distributed git projects
  • Collaboration
  • Code Review
  • Bugs
  • Project Tracking
  • Modern web interface + native OS applications

Two models for updating repositories:

  1. Shared repository - Everyone commits to same branch and everyone handles merging.
  2. Fork & Pull - Users fork (make a copy) of the repo and ask original owner to "pull" any changes back.
I mention this rather nitty-gritty detail to segway into the next slide. We currently only use the "Shared Repository" method.
